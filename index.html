Projects

Space Blaster

Space Blaster is a game created using C# and the GameMaker Studio 2 platform. In this game, the player controls a ship and must survive against an endless wave of enemies using their space blasters. Players can accelerate the ship using right-click, and shoot using left-click. The game implements elements such as momentum, a high-score tracker, endless-increase in difficulty, and more!

Play the game here!

Cave Boy

Cave Boy is a game created using C# and GameMaker Studio 2 platform. In this game, the player controls a "cave boy" that ventures through a perilous cave! This game implements platforming, ledge grabbing, a high-score system, various enemies, and a boss!

Visit the website here! (You must use OperaGx browser to play it!)

NoteFinderFree

NoteFinderFree is a website that allows students to search up any notes on any subject from any school, given that previous students have uploaded their notes onto the site. Students can upload their notes using a PDF parser that will translate the notes ihto text for others to search up. This website was made using React.js, Javascript, HTML, and CSS.

Visit the website here!

Covid-19 and Homeless People

This website showcases the effect that COVID-19 had on the American Population. In this website, users can see three graphs that convey number of homeless men vs. women, job loss rates, and homeless people encountered in hospitals. This website was made to place a spotlight on a group of people that are otherwise shadowed in society. In an effort to bring more attention to the homeless population recovery movement, I have also added a map that showcases various places that have emerged to help homeless people as a result of this shadowing (made with python, HTML, and CSS).
